# Alarm Clock Card - Lovelace Custom Card

A beautiful, feature-rich Lovelace custom card for the Alarm Clock integration.

## Features

- **Compact and expanded modes**
- **Quick time adjustments** (+5m, +10m, +1hr)
- **Day toggle pills** - Quick weekday selection
- **State-aware UI** - Different view when ringing
- **Large touch targets** - Mobile-friendly
- **Theme support** - Respects HA themes
- **Visual card editor** - No YAML needed

## Prerequisites

This card requires the Alarm Clock backend integration to be installed first.
Install the backend from the [alarm_clock_backend repository](https://github.com/fappsde/alarm_clock_backend).

## Quick Start

1. Install via HACS (Frontend section)
2. Add the card to your dashboard
3. Select your alarm entity
4. Enjoy your beautiful alarm clock card!
