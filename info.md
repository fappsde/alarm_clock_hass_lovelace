# Alarm Clock for Home Assistant

A robust, feature-rich alarm clock integration for Home Assistant.

## Features

- **Multiple independent alarms** - Create as many alarms as you need
- **Weekday selection** - Different schedules for different days
- **One-time alarms** - Auto-disable after trigger
- **Skip next** - Sleep in without disabling
- **Snooze** - Configurable duration and max count
- **Script integration** - Pre-alarm, alarm, post-alarm, and more
- **Reliability** - State persistence, missed alarm detection, health monitoring
- **Beautiful Lovelace card** - Mobile-friendly with theme support

## Quick Start

1. Install via HACS
2. Add the integration in Settings > Devices & Services
3. Create your first alarm
4. Add the Lovelace card to your dashboard
